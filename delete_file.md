# Удалить файл из заказа

Метод удаляет файл из заказа.

POST `https://apiamb.kosmoslogistic.ru/api?command=delete_file`

## Заголовки

| Заголовок      | Значение             |
| -------------- | -------------------- |
| Authorization  | `<your-token>`       |
| Content-Type   | application/json     |

## Параметры запроса

Параметры читаются из query/form:

| Параметр   | Обязательный | Описание                                  |
| ---------- | ------------ | ----------------------------------------- |
| `file_ref` | Да           | Идентификатор файла в заказе (`ref`). |

## Что делает метод

1. Находит файл по `file_ref` и проверяет, что файл принадлежит найденному клиенту.
2. Удаляет запись о файле.

## Успешный ответ

### 200 OK

```json
{
  "result": "OK"
}
```

## Ошибки

### 401 Unauthorized

Токен не был передан.

```json
{
  "error": "access_denied",
  "message": "Токен не задан."
}
```

### 403 Forbidden

Токен недействителен или файл не принадлежит клиенту.

```json
{
  "error": "access_denied",
  "message": "<текст ошибки проверки токена>"
}
```

```json
{
  "error": "error",
  "message": "Файл не найден!"
}
```

### 404 Not Found

Клиент, связанный с токеном, не найден.

```json
{
  "error": "error",
  "message": "Клиент не найден"
}
```

### 501 Not Implemented

Токен используется только для просмотра и не позволяет удалять файл.

```json
{
  "error": "error",
  "message": "Только просмотр!"
}
```
